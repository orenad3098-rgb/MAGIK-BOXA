# MAGIK-BOXA 🎉📱  
אפליקציה ניידת לחתונות, מסיבות ואירועים קהילתיים – עם מוזיקה, נושאים ואפקטים חיים.

## 📖 תיאור כללי  
MAGIK-BOXA היא "תיבת קסמים" דיגיטלית לאירועים חיים.  
המטרה: להפוך כל אירוע לחוויה אינטראקטיבית, דינמית ומלאת אנרגיה, באמצעות שילוב של:

- 🎵 מוזיקה  
- 🎨 נושאים (Themes)  
- ✨ אפקטים חיים (Live Effects)  
- 📲 אינטראקציה בזמן אמת עם הקהל  

האפליקציה מיועדת ל:
- חתונות  
- מסיבות  
- אירועים קהילתיים  
- כל מקום שבו רוצים לחבר בין מוזיקה, ויזואליה והקהל.

---

## 🎯 Use Cases

- האורחים מצביעים דרך הטלפון על אפקטים / שירים.  
- ה‑DJ / המארגן רואה בזמן אמת מה הקהל רוצה.  
- האפליקציה מפעילה אפקטים ויזואליים בהתאם לבחירות הקהל.  
- יצירת "Mood" לאירוע לפי נושא (רטרו, ניאון, אלגנט, פנטזיה וכו').

---

## 🚀 פיצ'רים מתוכננים

### 🎨 Themes  
חבילות עיצוב (צבעים, אנימציות, אייקונים) לפי סוג האירוע.

### ✨ Live Effects  
אפקטים ויזואליים/תנועתיים שמופעלים לפי טריגרים (בחירת הקהל, שינוי בשיר, רגעים מיוחדים).

### 🗳️ אינטראקציה עם הקהל  
הצבעות חיות, בחירת שירים, טריגר לאפקטים דרך הטלפונים של המשתתפים.

### 🎚️ מצב מארגן / מצב אורח  
- מארגן: לוח שליטה, ניהול נושא, אפקטים, צפייה בהצבעות.  
- אורח: חוויית משחק/שליטה פשוטה ומהירה.

---

## 🛠️ טכנולוגיות מוצעות

- **אפליקציה ניידת:** React Native + TypeScript  
- **Backend & Realtime:** Firebase (Auth, Firestore, Realtime listeners)  
- **ניהול מצב:** Zustand / Redux / Context API  
- **UI:** Styled Components / React Native Paper
